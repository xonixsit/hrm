Key Features

  • Role Assignment: Admins can assign roles (Admin, HR, Manager, Employee) to users
  • Role Removal: Admins can remove roles from users
  • Safety Checks: Prevents removing Admin role from the last administrator
  • User Overview: Shows all users with their current roles and employee information
  • Role Statistics: Dashboard cards showing role distribution
  • Access Control: Only Admin users can access role management

Available Routes

  • GET /admin/roles - Role management dashboard
  • POST /admin/roles/{user}/assign - Assign role to user
  • POST /admin/roles/{user}/remove - Remove role from user

How to Access

  1. Log in as an Admin user (admin@example.com or support@xonixs.com)

  4. Use the dropdown menus to assign roles to users
  5. Click "Remove" to remove roles from users
