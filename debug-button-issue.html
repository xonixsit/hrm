Debug Assessment Cycle Button Issue

Test Routes:

1. Direct URL: http://localhost:8000/assessment-cycles/create

2. Assessment Cycle Manager: http://localhost:8000/assessment-cycle-manager

Debugging Steps:

  1. Click the direct URL above to test if the create page loads
  2. Go to the Assessment Cycle Manager page
  3. Open browser developer tools (F12)
  4. Check the Console tab for any JavaScript errors
  5. Click the "New Assessment Cycle" button
  6. Check if any errors appear in the console

Common Issues to Check:

  • JavaScript errors in browser console
  • Route helper function not working
  • Inertia.js router issues
  • Button click event not firing
  • PageHeader component not handling href correctly
